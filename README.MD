# Instalação

Certifique-se de ter o Node.js instalado.

Clone esse repositorio.

```cmd
$ git clone

$ cd ./Whiteboard-Collaborative
```



## Instalação de dependências 

```javascript
$ npm install

Start the server

$ npm start
```
